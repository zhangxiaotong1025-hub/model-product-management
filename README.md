# 模型商品管理系统

基于设计文档实现的模型到商品全生命周期管理系统，提供完整的后台管理界面。

## 功能模块

### 1. 模型库管理
- 模型上传（单个/批量/Excel）
- 模型审核流程
- 模型状态管理（待审核、已上架、已下架、审核失败）
- 皮肤模型管理
- 模型搜索与筛选

### 2. 类目管理
- 前台类目设置
  - 类型库管理
  - 一二三级菜单管理
  - 关联类目配置
  - 关联模型配置
- 后台类目管理
  - 商品属性维护
  - 类目树结构管理
- 涂料类目支持
  - 色板显示
  - 颜色排序
  - RGB控件

### 3. 商品管理
- **平台商品管理**
  - 商品总数统计
  - 建模成功/失败统计
  - 上架/审核失败统计
  - 主数据管理
  - 子数据查看
  
- **企业商品管理**
  - 企业商品创建
  - 商品授权管理
  - 主数据/子数据维护
  - 商品关联模型

### 4. 数据看板
- 模型趋势分析
- 关键指标展示
- 高风险商品监控
- AI智能分析

## 技术特性

- 响应式设计，支持多端适配
- 现代化UI设计，参考PULSAR设计风格
- 数据可视化图表
- 实时搜索与筛选
- 批量操作支持
- 模态框交互

## 使用说明

1. 直接在浏览器中打开 `index.html` 即可使用
2. 系统包含模拟数据，可直接体验各项功能
3. 支持的浏览器：Chrome、Firefox、Safari、Edge（最新版本）

## 核心功能说明

### 商品数据性质
- **主数据**：自建商品数据，可被其他企业授权使用
- **子数据**：引用其他企业的商品，继承模型关系

### 权限管理
- **平台角色**：可管理所有企业的商品数据
- **企业角色**：仅可管理当前企业树下的商品数据

### 模型关联
- 支持关联类目和关联模型两种方式
- 模型去重显示
- 支持主模型和皮肤模型
- 实时状态同步
